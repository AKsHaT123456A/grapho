Sample Documents for Testing

This folder contains sample text documents you can use to test the Knowledge Graph Builder.

Files:
- sample1.txt: About Apple Inc. and tech industry
- sample2.txt: About Tesla and renewable energy

To test:
1. Create a new workspace
2. Upload these files
3. Watch the AI extract entities and relationships
4. Explore the interactive graph

Expected entities:
- People: Tim Cook, Elon Musk, Jony Ive, etc.
- Organizations: Apple Inc., Tesla, SpaceX, etc.
- Locations: Cupertino, Austin, California, etc.
- Dates: January 2024, February 2024, etc.
- Features: iPhone, Powerwall, Supercharger, etc.

The AI will automatically detect relationships like:
- Tim Cook WORKS_FOR Apple Inc.
- Apple Inc. LOCATED_IN Cupertino
- Elon Musk FOUNDED Tesla
- Tesla COMPETES_WITH Ford
